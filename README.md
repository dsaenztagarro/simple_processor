# SimpleProcessor

Rails data batch migrations made easy.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'simple_processor'
```

And then execute:

```shell
bundle
```

Or install it yourself as:

```shell
gem install simple_processor
```
